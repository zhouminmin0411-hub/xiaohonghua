# 🎨 《小红花·绘本风 UI 组件规范》

---

# 1. 设计基调 & 全局风格

## 视觉关键词

* 水彩肌理
* 柔和低饱和度
* 厚涂手绘边缘
* 儿童绘本式拟物
* 大圆角、高容错

## 布局规范

* iOS 16/17/18 Safe Area
* 小程序默认底部 TabBar 高度：49 px（iOS 实际物理为 83 px 带 Home Indicator）
* 所有可点击区域 ≥ 44×44 pt

---

# 2. 绘本风按钮规范（Primary / Secondary）

## 2.1 Primary Button（主按钮：任务完成）

### 样式说明

* 背景色：暖橙色渐变，水彩纹理
* 圆角：24 pt（胶囊型）
* 描边：1 pt，淡棕色
* 阴影：0/4/8/0 高斯扩散
* 文字：白色，SF Rounded Semibold

### 尺寸

* 高度：48 pt
* 最小宽度：160 pt
* 左右 Padding：20 pt

### 状态

| 状态       | 背景       | 文本     | 描边 | 其他        |
| -------- | -------- | ------ | -- | --------- |
| Normal   | 橙黄水彩     | 白色     | 有  | 默认        |
| Pressed  | 深橙压暗 10% | 白色     | 有  | 按压缩放 0.96 |
| Disabled | 灰橙 40%   | 白色 60% | 无  | 不可点击      |

---

## 2.2 Secondary Button（次按钮：轻操作）

* 透明背景 + 浅棕色描边
* 高度：40 pt
* 文字颜色：深棕色

---

# 3. 绘本风卡片规范（任务卡片）

## 3.1 样式结构

* 背景：淡奶黄 + 水彩纹理
* 圆角：20 pt
* 内阴影极弱
* 图标置左、按钮置右

## 3.2 内容布局

| 区域   | 内容        | 规范                        |
| ---- | --------- | ------------------------- |
| 左侧图标 | 手绘图标      | 宽高 44–56 pt               |
| 标题   | 任务名       | SF Rounded Semibold 18 pt |
| 副标题  | 完成就能得到小红花 | SF Rounded Regular 14 pt  |
| 右侧按钮 | 我做完啦      | Primary Button            |

### 间距规范

* 左右间距：16 pt
* 图标与文字间距：12 pt
* 卡片间距：12 pt

---

# 4. 绘本风图标规范

## 图标共性风格

* 水彩粗边缘（深棕线条 2 pt）
* 柔和体积感
* 低饱和度
* 任务图标尺寸：48×48 pt
* TabBar 图标尺寸：32×32 pt

## 图标清单

### 任务图标

* 牙刷：蓝色柄 + 乳白刷毛
* 书本：浅绿 + 米黄
* 玩具箱：棕色箱子 + 泰迪熊

### TabBar 图标

* 更扁平、细节更少
* 适配小尺寸显示

---

# 5. 绘本风底部 TabBar 规范

## 尺寸

* 小程序标准高度：49 px
* iOS 实机高度：83 px（含安全区）

## 样式

* 背景：浅橙渐变 + 水彩纹理
* 点击区域 ≥ 60×60 px
* 选中态图标加深 15%
* 文案：SF Rounded 12 pt

## Tab 项

1. 首页（任务）
2. 小红花（瓶）
3. 礼物屋

---

# 6. 许愿瓶组件五段状态

## 容器规范

* 玻璃质感水彩瓶
* 宽 240 pt / 高 260 pt
* 底部柔和阴影

## 状态定义

### 0%（空瓶）

* 无花朵、轻光感
* 文案：0/50

### 25%

* 10–12 朵花，覆盖瓶底 1/4

### 50%

* 20–25 朵花
* 随机倾斜 4–8°

### 75%

* 接近满瓶，颜色略加深

### 100%（满瓶）

* 全部填满
* 可加入轻闪光特效
* 文案：50/50
