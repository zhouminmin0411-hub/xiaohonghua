# 阿里云域名ICP备案指南 - 小红花项目

## 🎯 目标
为 `amyworkshop.com` 完成ICP备案，以便未来绑定到云托管

**预计时间：** 15-20个工作日  
**费用：** 免费（阿里云提供免费备案服务）

---

## 📋 备案前准备

### 1. 确认域名信息

- **域名：** `amyworkshop.com`
- **域名所有者：** 必须与备案主体一致
- **域名注册商：** 阿里云（已满足）
- **域名有效期：** 至少3个月以上

### 2. 准备备案资料

#### 个人备案（推荐，较快）

| 资料 | 要求 |
|-----|------|
| **身份证** | 正反面清晰照片 |
| **手机号** | 备案主体本人手机号 |
| **邮箱** | 常用邮箱 |
| **个人照片** | 白色背景，半身照 |

#### 企业备案（如有公司）

| 资料 | 要求 |
|-----|------|
| **营业执照** | 清晰照片或扫描件 |
| **法人身份证** | 正反面 |
| **法人手机号** | 法人本人手机号 |
| **企业座机** | 可选 |
| **法人照片** | 白色背景，半身照 |

### 3. 重要提示

⚠️ **备案主体选择：**
- **个人备案**：审核快（15天），适合个人项目
- **企业备案**：审核慢（20天），但更正式

⚠️ **备案地域：**
- 选择您的**身份证地址**所在省份
- 部分地区要求提供居住证明

---

## 🚀 备案流程

### 第一步：登录阿里云备案系统

1. **访问备案管理控制台**
   ```
   https://beian.aliyun.com/
   ```

2. **登录阿里云账号**
   - 使用购买域名的账号登录

3. **点击"开始备案"**

---

### 第二步：填写备案信息

#### 1. 选择备案类型

- **首次备案**：第一次备案
- **新增网站**：已有备案，新增域名
- **接入备案**：域名已备案，接入阿里云

**您的情况：** 选择 **首次备案**

#### 2. 填写主体信息

**个人备案示例：**
```
主体类型：个人
姓名：[您的真实姓名]
证件类型：身份证
证件号码：[身份证号]
证件地址：[身份证地址]
联系方式：[手机号]
电子邮箱：[邮箱]
```

**企业备案示例：**
```
主体类型：企业
单位名称：[营业执照上的公司名]
证件类型：工商营业执照
统一社会信用代码：[营业执照上的代码]
单位地址：[营业执照地址]
投资人或主管单位：[法人姓名]
主体负责人姓名：[法人姓名]
主体负责人证件号码：[法人身份证号]
主体负责人手机号：[法人手机]
```

#### 3. 填写网站信息

```
网站名称：小红花家庭任务管理（示例，需要规范命名）
网站域名：amyworkshop.com
云服务：选择"阿里云"
服务类型：选择"云托管"或"云服务器"
服务号：申请阿里云备案服务号
网站负责人：[姓名]
网站负责人手机：[手机号]
```

**⚠️ 网站名称命名规范：**
- **个人备案**：
  - ❌ 不能包含：公司、教育、医疗等词汇
  - ❌ 不能包含：行业、通用词汇
  - ✅ 可以使用：个人博客、个人展示、姓名
  
- **企业备案**：
  - ✅ 可以使用公司全称或简称
  - ✅ 可以体现业务内容

**推荐命名（个人备案）：**
```
选项1：XX的个人博客
选项2：XX的个人网站
选项3：XX的学习笔记
```

**推荐命名（企业备案）：**
```
选项1：[公司名称]官网
选项2：[公司名称]产品展示
选项3：小红花家庭管理系统
```

#### 4. 申请备案服务号

如果您有云托管服务：

1. **进入备案服务号管理**
   ```
   https://beian.aliyun.com/#/icp/server
   ```

2. **点击"申请"**
   - 选择您的云托管实例
   - 每个实例可申请5个备案服务号

3. **复制服务号**
   - 格式：`3axxxxxx`
   - 粘贴到备案信息中

---

### 第三步：上传资料

#### 1. 上传身份证照片

要求：
- 格式：JPG、PNG
- 大小：不超过4MB
- 清晰度：能看清所有文字
- 正反面分别上传

#### 2. 上传半身照

要求：
- **背景：** 纯白色背景
- **服装：** 避免白色上衣
- **表情：** 自然，五官清晰
- **格式：** JPG、PNG
- **大小：** 不超过4MB

**提示：** 可以在家用白墙拍摄

#### 3. 其他资料

根据要求上传：
- 域名证书（阿里云自动生成）
- 承诺书（在线签署）
- 其他证明材料（部分地区要求）

---

### 第四步：提交初审

1. **检查所有信息**
   - 姓名、证件号无误
   - 手机号、邮箱正确
   - 照片清晰完整

2. **签署承诺书**
   - 在线签名或上传手写签名

3. **提交初审**

---

### 第五步：等待审核

#### 1. 阿里云初审（1-2个工作日）

- **审核内容：** 资料完整性、准确性
- **通知方式：** 短信 + 邮件
- **如果退回：** 按要求修改后重新提交

#### 2. 工信部短信核验（收到后24小时内完成）

收到工信部短信后：

1. **点击短信中的链接**
2. **填写验证码**
3. **确认信息**
4. **提交核验**

⚠️ **重要：** 必须在24小时内完成，否则备案会被退回

#### 3. 管局审核（10-20个工作日）

- **各省时间不同：**
  - 上海、广东：约10-15天
  - 北京、浙江：约15-20天
  - 其他地区：约10-20天

- **通知方式：** 短信 + 邮件

---

## ✅ 备案成功后

### 1. 获取备案号

- **备案号格式：** 京ICP备2024XXXXX号
- **在阿里云备案系统查看**
- **保存备案号**

### 2. 网站底部添加备案信息

未来网站上线后，需要在页面底部添加：

```html
<a href="https://beian.miit.gov.cn/" target="_blank">
  京ICP备2024XXXXX号
</a>
```

### 3. 绑定域名到云托管

备案成功后：
1. 在云托管绑定 `api.amyworkshop.com`
2. 配置DNS解析
3. 等待SSL证书签发
4. 配置到微信公众平台
5. 更新小程序代码

---

## ❌ 常见退回原因

### 1. 资料问题

| 问题 | 解决 |
|-----|------|
| 照片不清晰 | 重新拍摄上传 |
| 身份证过期 | 更新身份证信息 |
| 背景不符合要求 | 使用纯白背景 |

### 2. 网站名称问题

| 问题 | 解决 |
|-----|------|
| 包含禁用词汇 | 修改为个人博客等 |
| 名称过于笼统 | 添加个人前缀 |
| 涉及特殊行业 | 改为个人网站 |

### 3. 主体信息问题

| 问题 | 解决 |
|-----|------|
| 联系方式不通 | 更新为有效手机号 |
| 地址不详细 | 补充详细地址 |
| 域名所有者不一致 | 过户域名到备案主体 |
