# 微信开发者工具快速开始指南

## 问题：找不到 app.json

如果遇到 "在项目根目录未找到 app.json" 错误，说明打开的目录不正确。

## 正确的操作步骤

### 方法一：打开 miniprogram 目录（推荐）

1. **打开微信开发者工具**

2. **点击"+"或"导入项目"**

3. **选择项目目录时，选择**：
   ```
   /Users/xila/xiaohonghua/miniprogram
   ```
   **注意**：是 miniprogram 目录，不是 xiaohonghua 根目录

4. **AppID设置**：
   - 如果没有AppID，选择"测试号"或"不使用云服务"
   - 项目名称：小红花

5. **点击"导入"**

6. **构建npm**（首次打开必须）：
   - 工具 → 构建 npm
   - 等待构建完成（会生成 miniprogram_npm 目录）

7. **开发设置**（开发阶段）：
   - 详情 → 本地设置
   - ✅ 勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"
   - ✅ 勾选"启用自定义处理命令"（可选）

8. **点击"编译"**

### 方法二：修改 project.config.json

如果已经打开了错误的目录，可以：

1. 关闭微信开发者工具
2. 直接在Finder中进入：`/Users/xila/xiaohonghua/miniprogram`
3. 重新用微信开发者工具打开这个目录

## 常见问题

### Q1: 提示"未找到 app.json"
**A**: 打开的目录错误，请打开 `miniprogram` 目录

### Q2: 页面显示"module xxx is not defined"
**A**: 需要构建npm，执行：工具 → 构建 npm

### Q3: 样式不生效
**A**: 检查是否构建了npm，Vant组件需要npm

### Q4: 找不到 @vant/weapp
**A**: 
1. 在 miniprogram 目录执行: `npm install`
2. 然后在微信开发者工具中: 工具 → 构建 npm

### Q5: 测试家长密码是什么？
**A**: 默认密码是 `0000`
